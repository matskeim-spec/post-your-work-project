## Projektbeschreibung
Das Bikeshare-Projekt ist eine Python-Anwendung, die es Benutzern ermöglicht, Informationen über Fahrradausleihsysteme in verschiedenen Städten zu analysieren. Die Anwendung bietet Einblicke in die Nutzung von Fahrrädern, einschließlich der häufigsten Start- und Endstationen, der durchschnittlichen Fahrtdauer und anderer nützlicher Statistiken.

## Installation
